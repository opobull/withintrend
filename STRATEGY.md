## Phase 1: 기초 인프라 (1주차) — 내가 직접 처리

### 1-1. Google Search Console 연동
- **목적:** 인덱싱 상태 확인, sitemap 제출, 검색 성능 추적
- **필요:** Search Console API 서비스 계정 접근 (기존 opobot-analytics 계정에 권한 추가)
- **🔧 주인님 필요:** Search Console에 opobot-analytics 서비스 계정 소유자/사용자로 추가
- **자동화:** 매일 인덱싱 안 된 URL 탐지 → URL Inspection API로 제출

### 1-2. 내부 링크 자동화
- **목적:** 구글 크롤링 효율 + 페이지 권위 분산
- **방법:** 각 포스트 하단에 같은 카테고리 관련 포스트 3~5개 자동 링크
- **자동화:** 스크립트로 기존 400+개 포스트 일괄 처리
- **🔧 주인님 필요:** 없음

### 1-3. SEO 메타 강화
- **목적:** 검색 결과 CTR 개선
- **방법:** description 누락/약한 포스트 → 검색 의도 맞춘 메타 재작성
- **자동화:** 스크립트로 일괄 감사 + 수정
- **🔧 주인님 필요:** 없음

## Phase 2: 콘텐츠 배포 채널 (2~3주차)

### 2-1. Pinterest 자동 핀 ⭐ (최우선)
- **왜:** recipe, fashion, DIY 카테고리 = Pinterest 핵심 트래픽 소스. 자동화 쉽고 리스크 낮음
- **방법:** 새 포스트마다 대표 이미지 + 제목 + 링크로 핀 생성
- **카테고리 매핑:** recipes→Food, fashion→Fashion, diy→DIY, travel→Travel, home→Home Decor
- **자동화:** Pinterest API로 크론 연동
- **🔧 주인님 필요:** Pinterest 비즈니스 계정 생성 + API 앱 등록 + 토큰 발급
- **기대:** Pinterest는 핀 수명이 길어서 (6개월+), 쌓이면 꾸준한 유입

### 2-2. Reddit 시딩 (신중하게)
- **왜:** 백링크 + 즉각적 트래픽. 구글이 Reddit 결과를 상위 노출하는 추세
- **방법:** 관련 서브레딧에 유용한 답변/글 + 자연스러운 링크
- **주의:** 스팸 감지 엄격 → 계정 카르마 쌓기 선행, 링크 비율 10% 이하
- **자동화:** 반자동 (내가 글 초안 → 주인님 확인 후 게시, 또는 계정 주면 내가 직접)
- **🔧 주인님 필요:** Reddit 계정 (기존 or 새로) + 카르마 좀 쌓기
- **기대:** 게시물 하나 히트하면 하루 수백~수천 방문

### 2-3. Medium 교차 게시
- **왜:** 높은 DA, 구글 인덱싱 빠름, canonical URL로 SEO 페널티 없음
- **방법:** 인기 카테고리 포스트를 Medium에 재게시 (canonical → withintrend.org)
- **자동화:** Medium API로 자동 게시
- **🔧 주인님 필요:** Medium 계정 + Integration Token
- **기대:** Medium 내부 트래픽 + 백링크 효과

## Phase 3: 콘텐츠 품질 전환 (4주차~)

### 3-1. 롱테일 키워드 타겟팅
- **현재 문제:** 포스트가 너무 일반적 (경쟁 높은 키워드)
- **전환:** "best air fryer" ❌ → "best air fryer for single person apartment under $50" ✅
- **방법:** Google Autocomplete + People Also Ask 크롤링 → 경쟁 낮은 키워드 발굴
- **자동화:** 키워드 리서치 스크립트 → 진화 시스템에 연동
- **🔧 주인님 필요:** 없음

### 3-2. 프로그래매틱 SEO 강화
- **방법:** 특정 패턴 포스트 대량 생성 (예: "[브랜드] [직종] review", "[도시] best [음식]")
- **현재 알바 리뷰 포스트가 이 패턴** — 이걸 더 확장
- **자동화:** 템플릿 + 데이터 기반 자동 생성
- **🔧 주인님 필요:** 없음

### 3-3. 진화 시스템 고도화
- GA4 실제 데이터 기반 진화 (✅ 완료)
- 검색 유입 키워드 분석 → 그 키워드 관련 포스트 강화
- "거의 1페이지" 키워드 발견 → 해당 포스트 업데이트로 밀어올리기
- **자동화:** Search Console API 데이터 연동
- **🔧 주인님 필요:** 없음 (Phase 1에서 연동 완료 후)

## Phase 4: 수익화 (2~3개월차)

### 4-1. Google AdSense
- **조건:** 양질의 콘텐츠 + 일정 트래픽 (공식 기준 없지만 ~월 1,000+ 방문 권장)
- **🔧 주인님 필요:** AdSense 신청 + 승인 후 코드 삽입
- **자동화:** 승인 후 ads.txt, 광고 위치 최적화는 내가 처리

### 4-2. 제휴 마케팅 (Affiliate)
- **대상:** Amazon Associates, 쿠팡 파트너스 등
- **방법:** 리뷰/추천 포스트에 제휴 링크 삽입
- **자동화:** 포스트 생성 시 관련 상품 링크 자동 삽입
- **🔧 주인님 필요:** 제휴 프로그램 가입
